## Project setup

### Edit .env

```
VITE_BASE_URL=
VITE_I18N_FALLBACK_LOCALE=en
VITE_I18N_LOCALE=en
```

```
npm install
```

### Compiles and hot-reloads for development

```
npm run dev
```

### Compiles and minifies for production

```
npm run build & npm run preview
```

### Lints and fixes files

```
npm run lint
```

### Formats files

```
npm run format
```
